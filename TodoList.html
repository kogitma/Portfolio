概要

やるべき作業を書き出し、作業を整理・可視化することができます。
ログイン機能を実装しており、ログインユーザーごとに自身で登録した内容が表示されるようになっています（一部を除く）。
管理権限機能により、管理者権限を持ったユーザーのみが変更可能な項目もあります。
実装した機能は概ね企業実習先で提示された仕様書に準拠しています。

開発環境

OS : Ubuntu
開発言語 : PHP, HTML, Javascript
DBMS : MySQL
フレームワーク : Laravel, Bootstrap
その他 : Docker, Visual Studio Code, A5:SQL

補足事項

Todoリストを構成する各管理機能 : TODO、アカウント、カテゴリ、ステータス
基本の画面構成 : ログイン、メニュー、一覧、詳細、編集、新規登録、登録確認
共通機能 : 検索、表示件数変更、新規登録、編集、削除
