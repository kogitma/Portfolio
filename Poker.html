ポーカー

シャッフルされたトランプの山札から5枚手札が配られます。
山札がなくなるまで複数回手札を交換することができます。
手札の5枚に該当するポーカーの役が表示されます。

開発環境

OS : Windows 10
開発言語 : Java
その他 : Eclipse

補足事項

作ろうと思えば意図した役を作れてしまうため、楽しいかはわからない。
